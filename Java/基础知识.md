```java
    StringBuilder str_bid = new StringBuilder();

    str_bid.append(1);

```

# Integer比较
比较Integer的时候，要使用Integer.intValue()方法
```java
    Integer x = new Integer(10);
    Integer y = new Integer(10);

    System.out.println(x == y);
// This will return false since they will check whether they refer to the same object.

    Integer x = 10;
    Integer y = 10;

    System.out.println(x == y);
// This will return true since they all refer to one object.
```

# 进制讲解
一个带符号位的原码<br>
## 原码
最高位为符号位，0代表正数，1代表负数，非符号位为该数字绝对值的二进制<br>
## 反码
正数的反码与原码一致；负数的反码是对原码按位取反，只是最高位（符号位）不变。<br>
## 补码
正数的补码与原码一致；负数的补码是该数的反码加1

| - | 5 | -5 |
| - | - | - |
| 原码 | 0000 0101 | 1000 0101 |
| 反码 | 0000 0101 | 1111 1010 | 
| 补码 | 0000 0101 | 1111 1011 |

# Bit操作
```java
    &, |, <<, >>, >>>

    // >> Example
    a = 5 = 0000 0101
    b = -10 = 1111 0110

    a << 1 = 0000 1010 = 10
    a << 2 = 0001 0100 = 20 

    b << 1 = 1110 1100 = -20
    b << 2 = 1101 1000 = -40

    // >>> Example
    c = 15 = 0000 1111
    d = -15 = 
```
## Left Shifts
### <<
0010 << 1  →  0100<br>
0010 << 2  →  1000<br>


## Right Shifts
### >>
如果是正数，高位补0；如果是负数，高位补1；<br>
1011 >> 1  →  1101<br>
1011 >> 3  →  1111<br>

0011 >> 1  →  0001<br>
0011 >> 2  →  0000<br>

### >>>
不论是正数还是负数，高位补0；<br>
1011 >>> 1  →  0101<br>
1011 >>> 3  →  0001<br>
