# Construct String by String Builder
```java
    StringBuilder str_bid = new StringBuilder();
    str_bid.append(0);
    str_bid.append(1);
    str_bid.append(2);
    str_bid.append(3);
    str_bid.append(4);
    System.out.println(str_bid.toString());
    // 1. Delete Char By Range, include start index but not include the end index.
    str_bid.delete(0,2);
    System.out.println(str_bid.toString());
    // 2. Delete Char By index
    str_bid.deleteCharAt(str_bid.length() - 1);
    System.out.println(str_bid.toString());
```

# Integer比较
比较Integer的时候，要使用Integer.intValue()方法
```java
    Integer x = new Integer(10);
    Integer y = new Integer(10);

    System.out.println(x == y);
// This will return false since they will check whether they refer to the same object.

    Integer x = 10;
    Integer y = 10;

    System.out.println(x == y);
// This will return true since they all refer to one object.
```

# LinkedList
```java
    LinkedList<Integer> test = new LinkedList<>();
    test.add(100);
    test.add(200);
    test.add(300);
    test.add(400);
    test.add(500);
    System.out.println(test.indexOf(100));
    // Delete the element which is 100
    test.remove((Integer) 100);
    System.out.println(test.indexOf(100));
    // Delete the element of which index is 0
    test.remove(0);
```
